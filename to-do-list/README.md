\# 🚀 Smart Task Manager Pro



A modern and feature-rich desktop task management application built with \*\*Python\*\*, \*\*Tkinter\*\*, and \*\*SQLite\*\*. It helps users organize tasks, track progress, manage deadlines, monitor productivity, and maintain detailed activity logs through an intuitive graphical interface.



\---



\## 📌 Features



\### ✅ Task Management



\* Add new tasks with detailed information

\* Edit existing tasks

\* Delete single or multiple tasks

\* Mark tasks as:



&#x20; \* Pending

&#x20; \* In Progress

&#x20; \* Completed

\* Set task priorities:



&#x20; \* High

&#x20; \* Medium

&#x20; \* Low

\* Assign categories

\* Add tags and notes

\* Track task completion progress



\### 📂 Categories



Built-in categories include:



\* Study

\* Work

\* Personal

\* Health

\* Shopping

\* Finance

\* Other



Each category is displayed with a unique icon for better visualization.

\## ⏰ Reminder System



The application automatically highlights:



\* Overdue Tasks

\* Tasks Due Today

\* Upcoming Tasks (within 3 days)



The reminder window provides quick visibility into urgent tasks.



\---



\## 📈 Progress Tracking



Each task supports:



\* Progress percentage (0–100%)

\* Visual progress bars

\* Overall project completion tracking



The application automatically calculates:



Overall Completion %

= (Completed Tasks / Total Tasks) × 100

\## 🗄️ Database Structure



\### Tasks Table



| Column       | Type    |

| ------------ | ------- |

| id           | INTEGER |

| title        | TEXT    |

| description  | TEXT    |

| category     | TEXT    |

| priority     | TEXT    |

| due\_date     | TEXT    |

| status       | TEXT    |

| created\_at   | TEXT    |

| completed\_at | TEXT    |

| tags         | TEXT    |

| progress     | INTEGER |

| notes        | TEXT    |



\### Activity Log Table



| Column     | Type    |

| ---------- | ------- |

| id         | INTEGER |

| action     | TEXT    |
