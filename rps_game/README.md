\# ⚔️ Rock Paper Scissors Deluxe



A feature-rich and visually enhanced Rock-Paper-Scissors game developed using \*\*Python\*\* and \*\*Tkinter\*\*. This project goes beyond the traditional game by introducing multiple game modes, AI difficulty levels, achievement tracking, animated visual effects, and a modern user interface.



\---



\## 🎮 Overview



Rock Paper Scissors Deluxe is a desktop GUI application that combines classic gameplay with modern design principles. The application features dynamic animations, intelligent CPU opponents, achievement systems, match history tracking, and responsive user interactions, providing an engaging gaming experience while showcasing advanced Python programming concepts.

\## 🚀 Getting Started



\### Prerequisites



\* Python 3.8 or higher



\### Installation



Clone the repository:



```bash

git clone https://github.com/shekhawatnigam-design/CODSOFT/Rock-Paper-Scissors-Deluxe.git

```



Navigate to the project folder:



```bash

cd Rock-Paper-Scissors-Deluxe

```



Run the application:



```bash

python game.py

```
