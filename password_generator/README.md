\# 🔐 Secure Password Generator



A modern and feature-rich Password Generator built using \*\*Python\*\* and \*\*Tkinter\*\*. This application helps users generate strong, secure, and customizable passwords with an intuitive graphical user interface.



\## 📌 Project Overview



The Secure Password Generator is designed to create highly secure passwords for personal and professional use. Users can customize password length, character types, generate multiple passwords at once, exclude specific characters, and even create pronounceable passwords.



The application also includes password strength analysis, clipboard support, password history tracking, theme switching, and file export functionality.



This project was developed as part of the \*\*CodSoft Python Programming Internship\*\*.

\## ✨ Features



\### 🔑 Password Generation



\* Generate strong random passwords

\* Custom password length (4–64 characters)

\* Generate multiple passwords simultaneously (up to 10)

\* Support for:



&#x20; \* Uppercase letters

&#x20; \* Lowercase letters

&#x20; \* Numbers

&#x20; \* Special characters



\### 🗣️ Pronounceable Password Mode



\* Generate easy-to-remember passwords

\* Alternating consonant and vowel patterns

\* Improved readability while maintaining security



\### ⚙️ Advanced Customization



\* Exclude ambiguous characters:



&#x20; \* `l`

&#x20; \* `I`

&#x20; \* `1`

&#x20; \* `O`

&#x20; \* `0`

\* Exclude user-specified characters

\* Customize password composition



\### 📊 Password Strength Meter



\* Real-time strength evaluation

\* Categories:



&#x20; \* Weak

&#x20; \* Medium

&#x20; \* Strong

&#x20; \* Very Strong

\* Animated strength indicator



\### 📋 Clipboard Support



\* One-click password copying

\* Keyboard shortcut support



\### 🕘 Password History



\* Stores recently generated passwords

\* Quick copy from history tab

\* Clear history functionality



\### 💾 Save Passwords



\* Export generated passwords to a text file

\* Simple and secure local storage



\### 🎨 Theme Support



\* Dark Mode

\* Light Mode

\* Instant theme switching



\### ⌨️ Keyboard Shortcuts



| Shortcut | Action            |

| -------- | ----------------- |

| Ctrl + G | Generate Password |

| Ctrl + C | Copy Password     |

| Ctrl + S | Save Passwords    |
